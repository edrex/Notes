Looking into replacing keyboard/mouse with.. something

:ASETNIOP eliminates the need for reaching a key. Each finger is effectively one key in a chording system. Any input device that can generate a stream of per-finger “activate”/“deactivate” events can be used, such as:

  • conductive fingertip gloves http://www.asetniop.com/blog/?p=244
  • VR controllers like valve’s index controller https://uploadvr.com/latest-valve-index-controller-firmware-brings-noticeably-improved-finger-tracking/
